# Homebrew-adbs

[![Build Status](https://travis-ci.org/ksoichiro/homebrew-adbs.svg)](https://travis-ci.org/ksoichiro/homebrew-adbs)

A Homebrew formula for [ksoichiro/adbs](https://github.com/ksoichiro/adbs).

## Install

```sh
$ brew tap ksoichiro/adbs
$ brew install adbs
```

## Upgrade

```sh
$ brew update
$ brew upgrade adbs
```
